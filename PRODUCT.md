# Product

## Register

product

## Users

The primary user is a local iPhone user browsing, saving, organizing, and reading image galleries and shared media in a personal library.

## Product Purpose

MyEHViewer provides a reliable local workflow for discovering galleries, reading images, preserving selected content, and reviewing personal library statistics without uploading private data.

## Brand Personality

Focused, familiar, and content-first.

## Anti-references

Avoid marketing-style composition, decorative effects, unfamiliar controls, and layouts that compete with the media being read.

## Design Principles

- Keep media as the primary visual focus.
- Use familiar iOS controls and interaction patterns.
- Keep equivalent workflows visually and behaviorally consistent.
- Preserve layout stability during loading and chrome transitions.
- Prefer compact, task-focused interfaces over decorative presentation.

## Accessibility & Inclusion

Preserve Dynamic Type, VoiceOver labels, system contrast behavior, and reduced-motion preferences across interactive surfaces.
