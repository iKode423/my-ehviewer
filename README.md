# MyEHViewer

MyEHViewer 是一个 iOS 阅读应用项目，目标是支持浏览 `https://e-hentai.org/` 上与搜索、图库详情和阅读相关的主要功能。

## 当前阶段

- 已初始化 SwiftUI iOS 工程骨架。
- App 内可见文案使用中文。
- 已建立基础搜索页、阅读器页和设置页占位。
- 已建立项目文档目录 `Docs/`。
- 已查证搜索页、图库详情页和图片阅读页的公开 HTML 结构。
- 已建立领域模型、HTTP 客户端、搜索/详情/阅读页解析器和中性 fixture 测试。
- 已实现搜索页闭环：首页/热门/关注/收藏浏览来源、关键词搜索、隐藏分类、高级搜索参数、缩略图列表和上下页。
- 搜索失败后可保留当前请求参数直接重试。
- 搜索结果展示分类、页数、上传者、发布时间和标签摘要。
- 已实现图库详情页：从搜索进入详情，展示封面、标题、分类、上传者、元信息、评分、标签和页面入口，并支持标签继续搜索和加载失败重试。
- 已实现阅读器基础闭环：从详情页进入图片页，加载当前图片，支持上一页、下一页、输入页码跳转、已知页码范围、重试和原图入口。
- 已实现本地书架：记录历史、收藏和阅读进度，不缓存远端内容资源。
- 已实现设置页的本地数据统计与清空入口。
- 已实现从图库详情继续上次阅读，并支持在已知页面入口之间按页码跳转。
- 已实现图库详情页缩略图分页加载，可按页或一次性补全后续页面入口。
- 已实现阅读器显示偏好：图片显示模式、缩放倍率和阅读背景可在阅读器与设置页调整。
- 已实现最近搜索词记录与一键复用。
- 已实现站点 Cookie 配置：本机 Keychain 保存，网络请求自动带上 Cookie header，用于访问需要站点会话的关注和收藏来源。

## 技术选择

- SwiftUI
- Swift Concurrency
- URLSession
- XCTest
- 最低系统版本：iOS 17.0

## 验证

```sh
xcodebuild -project MyEHViewer.xcodeproj -scheme MyEHViewer -destination 'platform=iOS Simulator,name=iPhone 17 Pro' test
```

更多验证记录见 `Docs/QA.md`。

## 开发约定

- 每完成一个明确阶段，先验证，再提交对应代码。
- 所有面向用户的 App 文案默认使用中文。
- 不在仓库内保存站点内容、用户凭据、Cookie 或抓取到的图片资源。
